# paperproof-papers

This repository contains LaTeX paper projects for PaperProof, including the
academic paper, yellow paper, and whitepaper.

## Protocol and Labs Identity

PaperProof Protocol refers to the open protocol layer and official deployed
protocol instances. PaperProof Labs refers to the originating team and
maintainer of the official interface, SDKs, reference indexer, documentation,
and brand identity.

The papers should keep this distinction clear:

- Use PaperProof Protocol when discussing the protocol design, object model,
  deployed contract instances, event model, governance mechanism, PPRF
  coordination layer, Sui integration, Walrus content layer, and third-party
  interoperability.
- Use PaperProof Labs when referring to the originating team, official
  publisher, official documentation steward, official interface maintainer, or
  project operator.

Academic and technical papers should avoid unnecessary branding. Logo assets
are not required in this repository unless a specific publication format asks
for them.
